## Overview

This design system, derived from a Linear.app interface, presents a dark, modern, and highly functional aesthetic. It prioritizes clarity and efficiency through a minimalist color palette, a single, versatile typeface, and a structured layout. The visual design is characterized by its deep, muted backgrounds, subtle borders, and judicious use of accent colors to highlight interactive elements and status indicators. The overall impression is one of a sophisticated, developer-centric tool designed for focused work.

**Key Characteristics:**
*   **Dark Theme Dominance**: Predominantly dark surfaces and backgrounds, with lighter text.
*   **Inter Variable Typeface**: Exclusive use of "Inter Variable" for all text, ensuring consistency and readability.
*   **Subtle Elevation**: Minimal use of shadows, primarily for interactive or focused elements.
*   **Rounded Corners**: Consistent application of `4px` and `12px` border radii for a soft, approachable feel.
*   **Functional Accent Colors**: A range of vibrant accent colors used sparingly for status, alerts, and key actions.
*   **Clear Hierarchy**: Achieved through font weight and size variations within the "Inter Variable" family.
*   **Responsive by Design**: Explicit breakpoints for mobile, tablet, desktop, and wide screens.
*   **Minimalist Borders**: Thin, low-contrast borders define UI regions without adding visual clutter.

## Typography

### Font Family
The primary typeface used across the interface is "Inter Variable". This modern, highly legible sans-serif font is used for all textual elements, providing a consistent and clean aesthetic. The fallback stack for "Inter Variable" is `Arial, Helvetica, sans-serif` for smaller text, and `"SF Pro Display", -apple-system, "system-ui", "Segoe UI", Roboto, Oxygen, Ubuntu, Cantarell, "Open Sans", "Helvetica Neue", "Linear Thai", sans-serif` for larger text, ensuring broad compatibility. Emoji characters are supported by `"--font-emoji": "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Segoe UI", "Twemoji Mozilla", "Noto Color Emoji", "Android Emoji"`.
