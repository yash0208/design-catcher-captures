## Recommended Stack

*   **shadcn/ui**: For foundational UI elements like buttons, inputs, navigation, and cards, providing a solid, accessible base with Tailwind CSS integration.
*   **Aceternity UI**: Specifically for the `Spotlight` effect to enhance the hero section's visual appeal and provide dynamic interaction.
*   **Framer Motion**: To implement scroll-triggered reveal animations and other subtle interactive transitions detected on the page.
*   **Tailwind CSS**: As the underlying utility-first CSS framework for layout, spacing, and custom styling, complementing shadcn/ui and providing flexibility for unique components.
