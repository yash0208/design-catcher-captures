## Implementation Notes

1.  **Project Setup**: Initialize a Next.js (or equivalent React) project with Tailwind CSS.
2.  **Install shadcn/ui**: Begin by installing the core `shadcn/ui` components.
    *   `npx shadcn@latest init`
    *   `npx shadcn@latest add button card input navigation-menu`
3.  **Implement Layout**:
    *   Create a `Header` component using `shadcn-navigation-menu` for the main navigation links.
    *   Structure the main content area using Tailwind CSS grid or flexbox for the job listings and filter sidebar.
    *   For the "Hero Section (Image Background)", create a simple `div` with a background image and appropriate padding/sizing using Tailwind.
    *   For the "Fixed Side Entry/Floating Elements", create a custom component and use Tailwind's fixed positioning utilities (e.g., `fixed`, `bottom-x`, `right-x`).
4.  **Integrate Components**:
    *   Use `shadcn-button` for all interactive buttons, customizing variants (primary, secondary, disabled) as per the design.
    *   Employ `shadcn-input` for search bars and any other text input fields.
    *   Utilize `shadcn-card` for displaying individual job listings and other content blocks, populating them with relevant data.
5.  **Add Marquee**:
    *   Install `Magic UI Marquee`: `npx shadcn@latest add @magicui/marquee`
